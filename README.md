# Asgard-LP-Alert Chat Bot

## PREREQ:
* have node installed

## INSTALLING:
* clone the repo
* install package dependencies
** coingecko-api
** discord.js
* create a token.js file that looks like the following: 
* `var token = "insert.discord.token.here"
exports.token = token`
* start the bot with "node index.js"
